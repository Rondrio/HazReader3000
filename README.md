# HAZReader3000

#### HOW TO USE :
##### Step 1:
   + Find HAZ-article that is locked behind a pesky paywall
##### Step 2 :
 + Copy the url from the locked HAZ-article
##### Step 3 :
 + Start HAZReader3000
 ##### Step 4 :
 + Paste link into HAZReader3000 and press return
 ##### Step 5 :
 + Open article.md 
 ##### Step 6 : 
 + Enjoy your free HAZ-article
